Obiekty

W JavaScript prawiewszystko jest obiektem!

function show() {
console.log(this.brand + " koloru " + this.color);
}
 
const car1 = {
brand : "Mercedes",
color : "czerwony",
showText : show
}
 
const car2 = {
brand : "BMW",
color : "czarny",
showText : show
}
 
car1.showText(); //"Mercedes koloru czerwony"
car2.showText(); //"BMW koloru czarny"

Destrukturyzacja obiektów

Destrukturyzacja pozawala rozłozyć obiekt na pojedyncze pola.

const car = {
brand : "Mercedes",
color : "czerwony"
}const { brand, color } = car;
const engine = {
capacity : 2,
fuel : "disel"
}const carWithEngine = {...car, ...engine};console.log(carWithEngine); // { brand: "Mercedes", color: "czerwony", capacity: 2, fuel: "disel"};
